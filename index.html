Tables

Element that is used to host and display data

Tag/Attribute Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<th> Table Head Cell Container holding headers within a table cell.
<tfoot> Table Foot The set of rows defining the footer in a table.
colspan Column Span Defines how main columns a td element should span.
rowspan Row Span Defines how mainy rows a td element should span.

Head

Element that is used to host a documents meta data

Tag/Attribute Name Description
<head> Head The wrapper element for HTML head.
<title> Title Text only element that defines page title for toolbars and search engines.
<style> Single Page Style Element used to define style information with css code.
<link> Link Defines the relationship between a document and an external source.
<base> Base Target Defines a default address or target for all links on a page

Lists

Element that hosts data and displays it in a bulleted order.

Tag/Attribute Name Description
<ul> Unordered List List with default markings as bullets.
<ol> Ordered List List where items will be marked in numerical order.
<li> List Item Wrapper for each element contained in a list.
<dl> Description List List where data can be a term or a description to the term.
<dt> Description Term Defines a term in a description list.
<dd> Description Descriptor Describes the term in a description list.

Backgrounds

Adding design to the background of an object

Tag/Attribute Name Description
background-color Background Color Specifies the background color of and element.
opacity Opacity Specifies the transparency of and object from 0.0 to 1.0.
background-image Background Image Specifies an image to use as the background of an element. By default, the image is repeated so it cover the element.
background-repeat Background Repeat Property used to modify the default repetition of an image.
repeat-x/-y Repeat Horizontally / Vertically Used in conjunction with background-repeat to repeat an image only horizontally or vertically
no-repeat No Repeat Used in conjunction with background-repeat to only display and image once.

Borders

Adding visible lines between elements, for both visual appeal and usability.

Tag/Attribute Name Description
border-style Border Style Specifies border as: dotted, dashed, solid, double, groove, ridge, inset, outset, none, or hidden.
border-width Border Width Specifies the width of the four borders by size (px, pt, cm, em, ect).
border-color Border Color Sets color of borders with in put of name "red", HEX "#ff0000",
RGB "rgb(255,0,0)", HSL "hls(0, 100%, 50%)", or transparent.
